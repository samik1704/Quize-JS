# Quize-JS
- HTML Structure
- Defines the quiz layout with elements like div, button, and p.
- Each question is displayed with two options: Yes and No.
- CSS Styling
- Provides a clean and attractive interface.
- Buttons are styled for hover effects and responsiveness.
- Layout ensures readability and user-friendly design.
- JavaScript Functionality
- Handles user input and tracks answers.
- Displays feedback (e.g., correct/incorrect, next question).
- Can calculate and show the final score at the end of the quiz.
